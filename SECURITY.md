# Security Policy

## Reporting a vulnerability

Please report security issues **privately** via GitHub's
[private vulnerability reporting](https://github.com/JorgeCostaMacia/skills/security/advisories/new)
rather than opening a public issue.

These skills are plain Markdown/JSON instructions for AI coding agents — they
contain no executable runtime — but if you spot anything that could lead an
agent to unsafe behaviour, the same private channel applies.
